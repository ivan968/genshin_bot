# 🎮 Genshin Stats — Telegram Mini App

Telegram Mini App для перегляду статистики Genshin Impact по UID.  
Використовує **Enka.Network API** — безплатний публічний API.

---

## Що показує додаток

- **Профіль гравця**: нікнейм, рівень AR, рівень світу, досягнення, прогрес Безодні та Театру
- **Вітрина персонажів**: всі персонажі з відкритого showcase в грі
- **Деталі кожного персонажа** (при натисканні):
  - Зброя (рівень, refinement, іконка)
  - Артефакти (5 слотів з іконками та main stat)
  - Бойова статистика (HP, ATK, DEF, CRIT Rate/DMG, EM, ER)
- **Посилання на paimon.moe** для детальної статистики круток

## 🚀 Як задеплоїти

### 1. Хостинг файлу
Потрібно захостити `index.html` на будь-якому HTTPS хостингу:

| Сервіс | Безплатно | Швидко |
|--------|-----------|--------|
| **GitHub Pages** | ✅ | ✅ |
| **Netlify** | ✅ | ✅ |
| **Vercel** | ✅ | ✅ |
| **Cloudflare Pages** | ✅ | ✅ |

#### GitHub Pages (найпростіше):
```bash
# 1. Створи репозиторій на github.com
# 2. Завантаж index.html
# 3. Settings → Pages → Deploy from branch → main → / (root)
# URL буде: https://YOUR_NAME.github.io/REPO_NAME/
```

#### Netlify (drag & drop):
1. Зайди на **netlify.com**
2. Перетягни папку з `index.html` прямо на сайт
3. Отримай URL типу `https://amazing-app-123.netlify.app`

---

### 2. Створення Telegram Bot + Mini App

```
1. Напиши @BotFather в Telegram
2. /newbot → задай ім'я та username
3. Збережи токен бота
4. /newapp → вибери свого бота
5. Вкажи URL свого хостингу
6. Готово! Міні-додаток прив'язано до бота
```

Або додай кнопку меню:
```
/mybots → вибери бота → Bot Settings → Menu Button → Edit Menu Button URL
```

---

### 3. Налаштування в грі

Щоб дані відображались:
1. Відкрий Genshin Impact
2. Натисни на свій профіль → **Character Showcase**
3. Додай до 8 персонажів у вітрину
4. Переконайся що вітрина **публічна**

## ⚙️ Технічні деталі

- **API**: `https://enka.network/api/uid/{UID}/` 
- **CORS Proxy**: `api.allorigins.win` (безплатний, для браузера)
- **Іконки**: `https://enka.network/ui/` (офіційні ресурси)
- Жодних серверів, жодних баз даних — все в браузері

### Про кручення (wishes)
Enka.Network **не має доступу** до історії круток — це закрита інформація.  
Для трекінгу круток є тільки [paimon.moe](https://paimon.moe/wish) через офіційний API HoYoverse.

## 🔧 Можливі проблеми

| Проблема | Рішення |
|----------|---------|
| "Гравець не знайдений" | Перевір правильність UID (9 цифр) |
| Немає персонажів | Відкрий showcase в грі |
| Помилка 424 | Сервери Genshin тимчасово недоступні |
| Помилка 429 | Зачекай кілька хвилин (rate limit) |
